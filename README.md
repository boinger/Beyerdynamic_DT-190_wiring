# Beyerdynamic_DT-190_wiring
Wiring information for the Beyerdynamic DT-190 (and DT-290) headset

## K190.NN cable wires
red = Right ear +
blue = Right ear −
orange = Left ear +
brown = Left ear −
bare = Mic shield
green = Mic + signal
yellow = Mic − signal

## Abbreviations used by Beyerdynamic (DIN spec)
* bl = blue
* br = brown
* gn = green
* or = orange
* rt = red
* ws = white
* ge = yellow
* Sch / Schirm = shield

### Other DIN abbreviations (for completeness)
* sw = black
* gr = gray
* rs = pink
* vi = purple
* tk = turquoise
